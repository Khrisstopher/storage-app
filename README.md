# Storage App

Aplicación web básica para la gestión de archivos con reglas de almacenamiento y validación.

## 📌 Descripción

Este proyecto consiste en el desarrollo de un sistema que permite a los usuarios subir archivos de forma segura, aplicando restricciones como:

* Límite de almacenamiento por usuario
* Restricción de tipos de archivo
* Validación de archivos comprimidos (.zip)

Incluye un sistema de roles con usuarios y administradores, así como una interfaz simple para la gestión de archivos.

---

## ⚙️ Tecnologías

* XAMPP (entorno local)
